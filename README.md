jekyll-boilerplate
==================
by Leon (l3kn) Rische 

Version 0.9.2

## Usage

###1. Install the following Gems
  
`gem install jekyll`  
`gem install sass`  
`gem install bourbon`  
`gem install jekyll-asset-pipeline`

###2. Change the _config.yaml

###3. run `jekyll`
